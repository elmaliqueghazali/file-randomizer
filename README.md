# File Randomizer

## Description

To generate files of a specific type and size (up to 1GB).

## Supported File Types

1. PDF
2. DOCX
3. PPTX
4. XLSX
5. TXT

## Installation

```
npm install file-randomizer
```

## Example

To generate a PDF of 200MB in size, run the following command:

```
file-randomizer --size=200 --type=pdf
```
